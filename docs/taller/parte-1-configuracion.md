# Parte 1 — Configuración del entorno

## Objetivo
Dejar Android Studio listo para crear, compilar y ejecutar una app Android con Kotlin.

## Qué se revisa
- Instalación de Android Studio y del SDK
- Creación de un proyecto con **Empty Views Activity**
- Elección de Kotlin como lenguaje
- Configuración del `minSdk`
- Uso de un emulador o dispositivo físico

## En este proyecto
- El módulo principal está en `app/`
- La actividad de arranque es `MainActivity.kt`
- El manifiesto se encuentra en `app/src/main/AndroidManifest.xml`

## Recomendación práctica
Antes de tocar código, confirmar que Gradle sincroniza sin errores y que el proyecto abre correctamente en Android Studio.
