# Parte 3 — Ejercicios de práctica

## Objetivo
Practicar Kotlin con problemas cortos antes de pasar al proyecto Android.

## Ejercicios del taller
1. **Null safety**: construir una función que describa un estudiante y use un valor por defecto cuando el correo sea `null`.
2. **`when`**: calcular descuentos según el tipo de cliente.
3. **Colecciones**: filtrar estudiantes aprobados, ordenarlos por nota y quedarse con sus nombres.
4. **Data classes**: calcular un resumen de carrito con cantidad total de productos y total monetario.

## Qué se aprende
- Cómo pensar en transformaciones de datos
- Cómo usar operaciones de colección encadenadas
- Cómo modelar objetos de forma simple con `data class`

## Sugerencia de trabajo
Resolver primero sin mirar la solución y luego comparar el resultado con la sintaxis del taller.
